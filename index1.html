Question-2:What is the difference between innerHTML, innerText, and textContent ?

textContents is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its child elements. innerHtml returns all text, including html tags, that is contained by an element.

Question-3:What is event delegation in the DOM?

a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element

Question-4:What is event bubbling in the DOM?

the behavior where when an event occurs on an element, it triggers not only on that element but also propagates upwards through the DOM hierarchy, firing event handlers on its parent elements until it reaches the topmost element (usually the document)
